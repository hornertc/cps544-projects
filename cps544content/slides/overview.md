## Course Caption

> The study of advanced topics related to software development in the UNIX/Linux environment. Topics may include advanced inter-process communication (IPC), systems administration, virtualization and security

----

## Modified Course Caption

<sub><sup>Prepares students for developing software in the UNIX/Linux environment using the Go programming language.</sup></sub>

<sub><sup>Possible topics include libraries and system calls, shells, system structures and internals, inter-process communication (pipes and signals), communicating sequential processes, network programming (client-server model and sockets), configuration and compilation management, pattern matching and filters, shell programming, automatic program generation, and GUI programming.</sup></sub>

----

## Course Content

- Assignments are designed to provide students with a pragmatic exposure to these tools as well as issues faced by modern practitioners.
- CPS 544 is a programming-intensive course
- Main course topics (in decreasing focus)
  - Go
  - POSIX shell
  - UNIX/Linux

----

## Prerequisites

- CPS 501 Advanced Programming and Data Structures
- Experience programming in any statically typed language (e.g., C/C++, Java, C#)
- Exposure to filesystems, concurrency, and multi-threaded systems is not required but preferred (e.g., a course in Operating Systems)
