- Linux vs UNIX vs BSD:
  - **Linux**: Open-source kernel used in distributions like Ubuntu, Fedora.
  - **UNIX**: Proprietary OS standard forming the basis for systems like AIX.
  - **BSD**: UNIX-like OS focused on security, also open-source.

- Sticky Bit:
  - Example: Set sticky bit on a directory:
    ```bash
    chmod +t /shared-directory
    ```
  - Ensures only file owners can delete files in `/shared-directory`.
